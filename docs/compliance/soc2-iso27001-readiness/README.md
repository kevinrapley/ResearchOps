# SOC 2 and ISO 27001 readiness

This folder holds the compliance-readiness evidence pack for the ResearchOps platform.

This evidence pack does not assert SOC 2 compliance or ISO/IEC 27001 certification. It defines the service boundary, proposed assurance scope and evidence still needed before the organisation can make any external or internal compliance claim.

## Purpose

The workstream supports future assessment against:

- SOC 2, using the AICPA SOC suite as the assurance family reference: <https://www.aicpa-cima.com/resources/landing/system-and-organization-controls-soc-suite-of-services>
- ISO/IEC 27001, using the ISO information security management system standard as the management-system reference: <https://www.iso.org/standard/27001>

## Readiness rule

Repository changes may describe readiness, preparation, evidence and control implementation. They must not use compliance, audit-complete or certification-complete wording for SOC 2 or ISO/IEC 27001 unless there is a current signed assessment, certificate or assurance report in the approved evidence store.
