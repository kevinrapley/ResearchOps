Before you start

You should know:

  • the name of the project
  • what the research needs to learn
  • which service phase the work relates to
  • who owns or supports the research

What you will do

  1. Define the project.
  2. Add stakeholders, objectives and user groups.
  3. Add project ownership and notes.
  4. Check your answers before creating the project.
