    Fieldnotes

    Capture ambient notes without leaving the participant. Type, press Enter, and the note lands in the session record with a timestamp. Shorthand prefixes set the category as you type.

    The framework sets the note categories and their shorthand prefixes.
    Set automatically by a shorthand prefix, or choose one here.
    Press Enter to save, Shift and Enter for a new line. Use #tags for themes and @refs for participants. Do not enter participant personal data unless it is necessary for the research record.

    How shorthand prefixes work

    Start a note with a prefix and a space to set its category, for example q for a quote in the Fieldnotes framework. The prefix is removed from the saved note. Notes without a prefix use the category selected above.
