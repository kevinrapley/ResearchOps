    Variables

    Variables personalise your guide using Mustache tags like {{participant.name}}. This editor stores variables as JSON in Airtable (“Variables (JSON)”).

    • Create with “+ Add”. Use short, semantic keys (e.g. version or session.date).
    • Edit values inline. Accepts text, numbers, booleans, or JSON.
    • Delete using the ✖ on a row (with confirmation).
    • Save variables posts JSON back to Airtable without changing your Markdown body.
