      Which sort type should I choose?

      Use an open sort to learn how participants naturally group and label concepts, typically early in discovery. Use a closed sort to test whether an existing structure works, typically to validate navigation. A hybrid sort sits in between: participants start from your groups but can add their own when nothing fits.
