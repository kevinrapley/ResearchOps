# Research Taxonomy Reference

Status: draft Sourcebook template.

Use this reference to apply consistent tags and categories when storing and retrieving research findings.

## Top-level categories

| Category | Description |
|---|---|
| User behaviour | How users act in context |
| User attitude | What users say and believe |
| Unmet need | Gaps between current and desired experience |
| Pain point | Friction or difficulty in current experience |
| Design opportunity | Specific area where design can improve outcomes |
| Policy implication | Finding with policy or compliance relevance |
