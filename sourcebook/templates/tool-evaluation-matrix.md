# Tool Evaluation Matrix

Status: draft Sourcebook template.

Use this matrix when evaluating research tools for procurement or adoption.

## Evaluation criteria

| Criterion | Weight | Description |
|---|---|---|
| Functionality | High | Does it support the required research methods? |
| Accessibility | High | Does it meet WCAG 2.1 AA? |
| Data security | High | Does it comply with data protection requirements? |
| Integration | Medium | Does it integrate with existing tools? |
| Cost | Medium | Is it within budget? |
| Support | Medium | Is vendor support adequate? |
| Ease of use | Low | Can the team adopt it without significant training? |

## Scoring guide

Score each criterion: 1 (poor) – 3 (acceptable) – 5 (excellent). Multiply by weight before summing.
