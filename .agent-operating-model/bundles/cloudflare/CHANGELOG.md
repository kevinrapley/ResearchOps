# CHANGELOG

## 1.0.0

Initial Cloudflare Developer Platform prompt bundle.

### Added

- Cloudflare bundle orchestration, prompt spec and schemas.
- Source catalogue using only `developers.cloudflare.com` URLs.
- Reference modules for Workers, Wrangler, bindings, storage, state, Pages, Queues, Workflows, Workers AI and Vectorize.
- Reference modules for deployments, versions, Cron Triggers, testing, observability, compatibility, limits, Service Bindings, Smart Placement, Hyperdrive and Static Assets.
- Mode modules for build, review and release work.
- Contract schemas for Wrangler configuration review, Cloudflare validation evidence and Cloudflare gap registers.
- Example Wrangler configuration and deployment evidence files.
- Source catalogue validation script.
- Role modules, templates, grader, regression tests and red-team tests.
